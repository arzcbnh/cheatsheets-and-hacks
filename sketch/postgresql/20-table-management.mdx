### Creating a new table

```sql
CREATE TABLE <name> (
    <column1> <properties>,
    <column2> <properties>,
    ...
);

CREATE TABLE weather (
    city             varchar(80),
    temperature      int,
    precipitation    real,
    date             date
);
```
